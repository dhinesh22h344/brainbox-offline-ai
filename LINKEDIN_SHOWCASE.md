## 4. Push this project to GitHub (one-time)

In Terminal (or your IDE terminal), from the project folder:

```bash
cd "/Users/dhineshs/Desktop/Offline Chatgpt Ios"

# Initialize git (if not already)
git init

# Add everything (respects .gitignore)
git add .
git commit -m "Brainbox: offline AI chat app for iOS and Android"

# Create a new repo on GitHub (github.com → New repository), then:
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git branch -M main
git push -u origin main
```

Replace `YOUR_USERNAME` and `YOUR_REPO_NAME` with your GitHub username and the repo name you created (e.g. `brainbox` or `Offline-Chatgpt-Ios`).
